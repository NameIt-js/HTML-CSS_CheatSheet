CSS Cheatsheet

What is CSS?

Cascading Style Sheets, is a language that is used to custumize how HTML elements show in a browser. CSS is capable of creating style clases in order to change the design of a web page

Similar to the physical characteristics of a person

CSS can be written in a inline style (Inside HTML) or in a separate CSS file, in this document we will focus on the separate file.

To use a separate CSS file in a certain HTML document it is needed to add this line into the HTML head tag:

<link href='path to the css file' rel='stylesheet' type='text/css'>

Example Structure

Let's simulate how a CSS should be applied in both a HTML file and CSS file:

For example lets say we wanna modify the font and size of the h1 heading. We will only need to attach the line below into the HTML file and then the following into the CSS file:

  h1{ /*Here we select the part of the HTML to be modified*/
    font-family: Arial; /*We specify the property and its value, here we change the font*/
    font-size: 20px; /*And here we change the size */
  } /*close of the modification

Selector Reference

Selector Description
* Selects all elements
.class Selects all elements with the specified class
.class1.class2 Selects all elements with both classes
.class1 .class2 Selects all .class2 elements which are inside an .class1 element
element Selects the specified element
element.class Selects the elements with the specified class
element1, element2 Selets all elements belonging to element1 and element2
element1 element2 Selects element2 elements inside element1
#id selects the unique element with the specified id property

Specifity

Browsers must decide which are the most relevant values to a CSS property in a certain element. A selector could be extremly general like the * selector or extremly specific like the id selector. Browsers will apply the most specific selector that they found, if a certain element has two selectors with the same value Browsers will aply the last one that they found.

For example:

body{
  font-family: Helvetica;
}
h1{
  font-family: monospace;
}

Here the applied value will be the h1 selector, however, outside the h1 everything inside the body tag will have a Helvetica font family.

Basic Propierties

Font

Property Description Values
font-family Specifies the font family for the text family-name, generic-family, inherit
font-size Specifies the font size of the text length, %, inherit
font-style Specifies the syle for the font normal, italic, oblique, inherit
font-weight Specifies the weight for the font normal, bold, bolder, lighter, numerical and inherit

Text

Property Description Values

text-align Aligns the text inside the element left, right, center, justify
text-transform Transforms the letters inside the element none, capitalize, uppercase, lowercase

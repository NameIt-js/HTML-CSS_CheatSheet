HTML Cheatsheet

What is HTML?

HTML (HyperText Markup Language) is a lenguage used to structure and give content to a website.

Similar to the skeleton on the human body

This language is composed by tags, formed by two parts the opening and the closing tag the content will be inside them. For example:

opening tag--> <html> content </html> <-- closing tag

Basic Structure

An HTML basic structure is composed of the following elements

<!DOCTYPE html> <!-- Specify the document type (HTML5) to the browser-->
  <html> <!--The web page will be inside this tag-->
    <head> <!--The metadata (info needed by the browser not by the user) will be here -->
    <head>
    </body> <!--The visible content of the web page will be here-->
    </body>
  </html>

Semantic Tags

These tags describe the content of themselves, making the code more readable and easy to maintain.

Tag Description
<article> An independent part of the document that makes sense on its own.
<aside> Extra information indirectly related to the document.
<div> Generic container.
<figure> Represents self-contained content. It supports in <img> and <figcaption>.
<figcaption> Describes a caption.
<footer> Usually contains information about the author, copyright or navegation links related to the main content.
<header> It contains the introduction of a section.
<main> Used to point out the main content of the web page.
<nav> Provides navegation links, either to the current document or external documents.
<section> A generic independent part of the document which can not be related to other semantic tag.
<title> Goes into the head element and displays the title into the browser title bar.

Common Tags

Tag Description Atributes needed
<a> Anchor tag creates a hyperlink to the referenced url. href
<audio> Adds an audio to the web page. src
<button> Creates a button. type
<embed> Embeds external content to the selected point of the document. type, src, width, heigth
<h1>...<h6> Are heading elements with six levels.Higher the level bigger the heading.
<img> Embeds an image to the selected point of the document. src, alt
<p> The content between this tag represents a paragraph (block of text).
<span> Inline container of text used to group elements for styling.
<video> Embeds a video to the selected point of the document. controls width
<source> Specifies multiple resources for the audio,caption or video elements. src, type
<ol> Creates an ordered list. It leans in the <li> tag.
<ul> Creates an unordered list.It leans in the <li> tag.
<li> Creates an element of a list.

Other Tags

Text modifing Tags

Tag Description
<b> Used to bring attention to the content. Usually turns it bold.
<br> Produces a break line.
<em> Emphasis text and normally turns it into italic
<strong> Indicates important content. Turns it into bold text.
